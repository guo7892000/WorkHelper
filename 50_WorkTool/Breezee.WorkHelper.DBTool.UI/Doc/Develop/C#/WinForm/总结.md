### 关于VS 2022 + .net 6.0生成目录说明：						
所有解决方案下的项目（包括类库项目和WinForm项目），我都设置其生成目录为：					
E:\70_Env\71_CSharpProject\50_Breezee\EasyDB\Bin\					
但生成后，其下有一个Debug目录，但该目录下有两个目录：					
	\Debug\net6.0				
	\Debug\net6.0-windows				#WinForm运行时的目录  
需要在WinForm项目中引用其他所有项目，否则其没有引用的项目，不会生成到\Debug\net6.0-windows目录，从而导致IOC找不到DLL而报错！！					
						
### SplitContainer 拆分比例						
在C#中，使用SplitContainer控件可以创建可分割的界面，允许用户通过拖动分隔条来调整两个窗格的大小。SplitContainer控件有两个重要的属性，					
SplitterDistance和SplitterIncrement，可以用来控制分割比例和分割条移动的增量。  					
splitContainer1.SplitterDistance = splitContainer1.Width / 2;	
splitContainer1.Panel1Collapsed = true; //设置是否折叠左边或上边的区域，即隐藏。  
						
### 不同网格显示或隐藏右键某个菜单
```
private void dgvSelect_MouseDown(object sender, MouseEventArgs e)

	tsmiAddCodeName.Visible = true; //显示【加入编码名称】右键菜单

private void dgvInput_MouseDown(object sender, MouseEventArgs e)
```
