## Windows批处理示例
### 复制文件到两个不同目录：用到字符拼接
```
	::说明：%~dp0 是盘符加路径，即该当前批处理文件.bat文件所在的目录。注=赋值前后不要加空格！！					
	::本脚本是将MCC中几个主工程的DLL复制到子模块的目录中，以支持调试！					
	set preDir="C:\ProgramData\Alstom\Monitoring\CareCentre\Modules\"					
	set endDir="\Configuration\"					
	set CareCentre=%preDir%CareCentre%endDir%					
	set Interface=%preDir%Interface%endDir%					
						
						
	::1、CareCentre					
	set nowDir=%CareCentre%					
	xcopy /Y "%~dp0"Alstom.Monitoring.Configuration.Controls.dll %nowDir%					
	xcopy /Y "%~dp0"Alstom.Monitoring.Configuration.Interfaces.dll %nowDir%					
	::2、Interface					
	set nowDir=%Interface%					
	xcopy /Y "%~dp0"Alstom.Monitoring.Configuration.Controls.dll %nowDir%					
	xcopy /Y "%~dp0"Alstom.Monitoring.Configuration.Interfaces.dll %nowDir%					
	pause					

```	
### 关于暂停和忽略暂停：							
	当我们执行这个run.bat文件时，运行到pause会提示需要按任意键继续，并且命令行画页会停止，“暂停结束”不会打印出来。						
	如果在执行run.bat时，使用 “ run.bat < nul” ，会一次执行完，不会出现画页停止情况。						
							
### 批处理文件中调用批处理文件：							
	CopyFile_All.bat						
	::如果在执行run.bat时，使用 “ run.bat < nul” ，会一次执行完，不会出现画面停止情况。						
	::Addin						
	call CopyFile_Addin_Alarm.bat  < nul						
	call CopyFile_Addin_Generic.bat  < nul						
	::pause						
	pause						
							
	CopyFile_Addin_Alarm.bat						
	::说明：%~dp0 是盘符加路径，即该当前批处理文件.bat文件所在的目录。注=赋值前后不要加空格！！						
	::根据开发调试连接的服务器版本相应调整以下目录，目前是以10.8.79.180作为开发服务器。						
	set addinDir="%~dp0"Addins\Alarm.1.99.7\						
							
	::1、从生成目录复制文件到插件目录，以支持PC启动调试						
	::DLL						
	xcopy /Y "%~dp0"Alstom.Monitoring.Module.Presentation.Alarm.AlarmViewer.dll %addinDir%						
	xcopy /Y "%~dp0"Alstom.Monitoring.Module.Presentation.Alarm.UserCustomControl.dll %addinDir%						
	::Help						
	xcopy /Y "%~dp0"Help\Alstom.Monitoring.Module.Presentation.Alarm.AlarmViewer.en-US.chm %addinDir%Help\						
	xcopy /Y "%~dp0"Help\Alstom.Monitoring.Module.Presentation.Alarm.AlarmViewer.zh-CN.chm %addinDir%Help\						
	::pause						
	pause						
							
	CopyFile_Addin_EWV.bat						
	xcopy /Y "%~dp0"Languages\EWV.en-US.xaml %addinDir%Languages\						
	xcopy /Y "%~dp0"Languages\EWV.zh-CN.xaml %addinDir%Languages\						
							
	::pause						
	pause
