## SQL Server Management Studio概述
是用于管理SQL Server基础架构的集成环境。Management Studio提供用于配置、监视和管理SQL Server实例的工具。 此外，它还提供了用于部署、监视和升级数据层组件（如应用程序使用的数据库和数据仓库）的工具以生成查询和脚本。
## 推荐度
强烈推荐，它是一个用于管理 SQL Server 对象的功能齐全的实用工具，其中包含易于使用的图形界面和丰富的脚本撰写功能。

## 功能
很多人都把SSMS用作查询工具，但其实它的功能要丰富得多。通过SSMS，你可以在单一服务器中运行查询程序，也可以从注册服务器窗口中选择一个文件夹并点击“新的查询”，在多台服务器中进行查询。在同一个文件夹中，查询任务可一次在所有服务器上完成。另外，SSMS还有调试程序的功能，可以在服务器中逐步调试代码、检查变量并验证路径。但注意不要在生产服务器上用。
