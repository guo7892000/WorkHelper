## Navicat概述
Navicat 是一套可创建多个连接的数据库管理工具，可以方便地管理MySQL、Oracle、PostgreSQL、SQLite、SQL Server、MariaDB和 MongoDB等不同类型的数据库，它与阿里云、腾讯云、华为云、Amazon RDS、Amazon Aurora、Amazon Redshift、Microsoft Azure、Oracle Cloud 和 MongoDB Atlas等云数据库兼容。Navicat提供了GUI，即图形用户界面，可以通过安全且简单的方法创建、组织、访问和共享信息。
## 推荐度
不推荐，没有社区版，收费软件，查得很严，一般公司不允许安装，除非购买了版权。

## 产品系列
【Navicat Premium】：是一个可多重连接的数据库管理工具，它可让你以单一程序同时连接到 MySQL、MariaDB、MongoDB、SQL Server、Oracle数据库 、PostgreSQL和 SQLite 数据库，让管理不同类型的数据库更加方便。  
【Navicat for MySQL】：是一套专为 MySQL 和 MariaDB 设计的高性能数据库管理及开发工具。它可以用于任何版本 3.21 或以上的 MySQL 和 MariaDB 数据库服务器，并支持大部份 MySQL 和 MariaDB 最新版本的功能，包括触发器、存储过程、函数、事件、视图、管理用户等。  
【Navicat for MariaDB】：为 MariaDB 数据库管理和开发提供了一个原生环境，能连接本地或远程的 MariaDB 服务器，并与 Amazon RDS 和腾讯云兼容。  
【Navicat for MongoDB】：高效图形用户界面，用于管理和开发 MongoDB 数据库。它能连接本地或远程的 MongoDB 服务器，以及与 MongoDB Atlas、阿里云、腾讯云和华为云兼容。为管理、监控、查询和可视化数据提供了卓越的功能，轻松提高日常 MongoDB 操作的效率。  
【Navicat for Oracle】：是一套专为 Oracle 设计的强大数据库管理工具。它可以用于 Oracle 8.1 或以上 （macOS 版由 Oracle 9i 开始）Oracle数据库服务器，并支持大部份 Oracle 最新版本的功能，包括目录、表空间、同义词、实体化视图、触发器、序列、类型等。  
【Navicat for SQLite】：是一套专为 SQLite 设计的强大数据库管理及开发工具。它可以用于任何版本 2 或 3 的 SQLite 数据库，并支持大部份 SQLite 的功能，包括触发器、索引、视图等。  
【Navicat for SQLServer】：是一套专为 SQL Server 设计的高性能数据库开发及管理工具。它可以用于 SQL Server 2000、2005、2008R2 及 SQL Azure，并支持大部份 SQL Server 的功能，包括触发器、索引、视图等。  
【Navicat for PostgreSQL】：是一套专为 PostgreSQL 设计的强大数据库管理及开发工具。它可以用于任何版本 7.5 或以上的 PostgreSQL 数据库服务器，并支持大部份 PostgreSQL 最新版本的功能，包括触发器、函数、管理用户等。  
【Navicat for Redis】： 是一个易于使用的界面，用于可视化和优化 Redis 数据。与仅使用文本命令相比，它提供了一个组强大的功能，可以使常规管理任务变得更简单、更容易、更高效。它能连接到任何本地或远程 Redis 服务器，并与云数据库兼容，如 Redis Enterprise Cloud、Amazon ElastiCache 和 Microsoft Azure。  
【Navicat Premium Lite 免费版】：是 Navicat 的精简版，它包含了用户执行主要的基本数据库操作所需的核心功能。它允许你同时连接到各种数据库平台，包括 MySQL、PostgreSQL、SQL Server、Oracle、MariaDB，以及 Redis 和 MongoDB 等NoSQL 数据库，所有这些操作都可以通过一个应用程序来完成。此外，Navicat Premium Lite 兼容 GaussDB、OceanBase 和 PolarDB 等数据库。  
【Navicat Cloud】：为你的团队提供一个中央空间存储你的 Navicat 对象进行协作。在任何地方和设备上都能很轻松地与同事共享你的文件。在你的所有设备（包括 Windows、macOS、Linux 和 iOS）上同步连接设置、查询、模型、代码段、虚拟组信息，甚至是图表工作区。储存在 Navicat Cloud 中的文件会自动显示在 Navicat 桌面版和移动版上，这样你就能随时随地实时访问他们。
