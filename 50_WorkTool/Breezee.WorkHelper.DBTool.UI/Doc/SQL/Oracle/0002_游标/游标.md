### 游标的FOR循环
```
FOR DR IN (SELECT M.MAIN_BRAND_CODE,M.HELP_BRAND_CODE,M.ORDER_BRAND_CODE
            FROM T_DB_BRAND_CONFIG M WHERE M.IS_ENABLE = '1'
            AND M.BRAND_CONFIG_ID = V_BRAND_CONFIG_ID) LOOP
    IF NVL(V_RESULT, '-1') = '-1' THEN
        V_RESULT := DR.HELP_BRAND_NAME;
    ELSE
        V_RESULT := V_RESULT || '、' || DR.HELP_BRAND_NAME;
    END IF;
END LOOP;
```

    Fetch curBalance

	  begin
		/*逻辑处理，此处略*/

		  commit;
		WHEN OTHERS THEN
