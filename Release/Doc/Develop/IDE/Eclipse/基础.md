### 配置保存动作自动清理无效的import
进入 ‌Window > Preferences > Java > Editor > Save Actions‌  
勾选 ‌Perform the selected actions on save‌  
勾选 ‌Organize imports‌，保存后每次保存文件时自动清理导入。  

### 关于Eclipse 编译
通常我们选中【Build Automatically】后，修改代码并保存时，会自动编译。  
另外，在获取他人的代码后，IDE也会自动构建。  
取消勾中自动编译后，【Build Project】可以点击来编译。  
但也会存在自动构建时我们中断了，然后重新打开不会触发编译。即使我们手工取消了自动编译，手工编译也不触发，可以通过以下操作来实现：  
Project > Clean…，记得勾中【Start a build immediately】。
