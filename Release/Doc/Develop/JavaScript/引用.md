### 返回指定目录示例
```
问题：例如soso.com/1/2/3/4/5.html 我如何在5下直接回到2下，2下有多个目录，怎么写代码才能准确的回到了2。
答案：
【./】到4，【../../】到3，【../../../】到2。比如你在5.html这个目录下引用一个图片这个图片放在2，你可以这样：<img src='../../../2.jpg' /> 你也可以这样<img src='./../../../2.jpg' /> 
每【../】返回上一级，返回多级就多一个【../】。 【./】表示当前目录，【../】上级目录，【./../../】当前目录的上级的上级目录，依次类推 ./../../相当于../../。
```
