## WPF-示例
### 设置图标1
```
1、设置窗体图标：
<Window x:Class="WorkTool.WPF.StartUp.WMainForm"
        xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
        xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml"
        Title="WMainForm" Height="410" Width="637" Loaded="Window_Loaded" Closing="Window_Closing" Icon="12.ico">

2、设置按钮图标：
<Button Name="CloseChild" DockPanel.Dock="Right" Width="22" Height="20" Click="CloseChild_Click">
                    <Button.Content>
                        <StackPanel Orientation="Horizontal">
                            <Image Stretch="Fill" Source="./Icon/Delete_16.png" />
                        </StackPanel>
                    </Button.Content>

                </Button>

3、设置菜单项图标：
menutItem.Icon = new System.Windows.Controls.Image 
       { 
           Source = new BitmapImage(new Uri("images/sample.png", UriKind.Relative)) 
       };
	
```
